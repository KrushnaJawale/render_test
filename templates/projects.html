Library Management System

Console Application • C++ • OOP

Source Code

Overview

A comprehensive console-based application designed to streamline library operations. It replaces manual record-keeping with a digitized system using C++ File Handling. The system allows librarians to manage book inventories and student borrowing records efficiently, ensuring data persistence even after the program terminates.

Technology Stack

C++ (std17) File Handling (fstream) OOP Principles Data Structures

Key Features

  • Admin Authentication: Secure login for librarians to prevent unauthorized access.
  • Book Management: Add, update, delete, and search for books by ID or Title.
  • Issue/Return Logic: Automatically calculates due dates and late fines ($1/day).
